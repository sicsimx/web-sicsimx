¿Que es un navegador web?

Google Chrome es el navegador mas usado de la actualidad

Se entiende por navegador a todo aquel software, aplicación o programa informático qué realiza una interpretación de los distintos tipos de archivos o de la información de la web para que puedan ser vistos desde nuestros dispositivos.
Los navegadores web hacen uso de motores de búsqueda para encontrar mejores resultados en la web, motores de búsqueda como DuckDuckGo o Google.

Entre algunas de las funciones qué ofrece un navegador se encuentran:

- Visualizar páginas webs
- Lectura de archivos locales con extensiones como .pdf o .html
- Impresión directa de las páginas web
- Reproducción de archivos multimedia como videos o música.

Los navegadores web más populares son los siguientes:


• Mozilla Firefox: desarrollado por la fundación Mozilla, es de código abierto y no busca lucrar con la información de tu navegación.

• Google Chrome: desarrollado por Google, su código es cerrado y utiliza tus datos estadísticos para mejorar sus productos y servicios con fines lucrativos.

• Microsoft Edge: desarrollado por Microsoft, su código es cerrado y fue desarrollado para reemplazar a Internet Explorer en el sistema operativo Windows.

• Internet Explorer: es el navegador de compatibilidad de Microsoft, actualmente descontinuado y reemplazado por Microsoft Edge. Parte de su código es Open source y cerrado, fue desarrollado para el sistema operativo de Windows 95 y se mantuvo en las siguientes versiones.

• Safari: desarrollado por Apple en sustitución al navegador de Microsoft Internet Explorer, su código es cerrado y es navegador exclusivo se los sistemas MacOS e iOS.

• Opera: desarrollado por Opera Software, está hecho en parte de código abierto, aunque es privado su código, fue desarrollado con fines de lucro.

• Vivaldi: este navegador fue desarrollo por Vivaldi Technologies y es de tipo freeware, orientado a todas las personas interesadas en Tecnología. Su eslogan es "un nuevo navegador para nosotros y nuestros amigos".

¿Cómo elegir un buen navegador?


Algunos navegadores usan gran cantidad de recursos de tu equipo, por lo que es recomendable usar los más óptimos en tus dispositivos para evitar dolores de cabeza innecesarios. Usar un buen navegador puede optimizar tus tiempos de trabajo, puede ofrecerte privacidad o puede ofrecerte una buena interfaz personalizable, por ello es bueno usar varios de ellos para posteriormente escoger el qué será tu navegador predeterminado o favorito.

¿Y tú que navegador usas?

Los navegadores ofrecen bastantes opciones a explotar para sus usuarios tales como:

• Extensiones
• Barra de marcadores
• Personalización
• Guardado de contraseñas por usuario
• Limpieza de historial
• Notas
• Modo incógnito
• Compartir
• Modo lectura
• Sincronización
• Lectura de PDF
